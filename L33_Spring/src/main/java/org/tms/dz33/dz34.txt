    В домашнее задание по скачкам необходимо добавить аспект,
который будет выполняться вокруг метода, который запускает гонку (охватывает 5 кругов одного заезда).

    Логика должна рассчитывать время заезда и выводить информацию в консоль

    @Configuration
    @EnableAspectJAutoProxy

    @EnableScheduling                       (в конфигурацию установить для включения планировщика)
    @Scheduled(cron = "*/1 * * * * *")          (установить над тем что надо вызывать в отдельном потоке для выполнения)

    @EnableAsync        (в конфигурацию установить для включения создания отдельного потока)
    @Async          (над чем создать отдельный поток)

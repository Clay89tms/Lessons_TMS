Выучить новые аннотации
// Controller       - аннотация для обозначение класса контроллером который будет обрабатывать запросы и отклики от UI
// RequestMapping, GetMapping, PostMapping ,......      - аннотации обозначающие чем обрабатывать запросы и отклики
// RequestParam RequestHeader CookieValue           - сулат для получения или отклика параметра хедера и куков
// PathVariable             - часть пути которую получаем от строки

Поднять проект с любым эндпоинтом, убедиться, что эндпоинт отзывается через браузер или постман
